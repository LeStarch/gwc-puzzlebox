What Just Happened

This is what is known as an injection attack. Here we take some form of code and inject it into a system that processes that code. In doing this, we tricked the program to run that code. This can be very dangerous.

Our pusheen lives on another machine, not the one running this browser. Imagine this pusheen was a zero-size image. The user would not know it is there, but the computer hosting the image would still get a request for it and thus know about our user. This is one way you are tracked on the internet. Advertisers place zero-sized images in your page, and when the image loads, their server gets notified. Then they can track you....clever.

Where can this be dangerous? Think about social media. What if you could inject code (don't try it) into a social media post. Then anyone seeing that post could be tracked (like our image) or would be tricked into running code they didn't ask to run. You could tell their computer what code to run.

How do we prevent this on our sites? Rule 1 of security is to always sanitize user input. This means to ensure it is not dangerous. For HTML, that means removing all > or < symbols. Without these symbols, no tag can be used, and thus it is not possible to inject HTML. For each user input, think about what could be run if the input was some type of code and then prevent that code by stripping out needed characters.

A Bit of HTML

A webpage, at its most basic, is text and special annotations called tags. Text and the tags are called HTML. That means everything you are reading here is HTML as drawn (rendered) by your browser. If you want to see this HTML, right-click and select "View Page Source". This will display the text along with the tags used to change its rendering.

HTML tags are words inside < > pairs. These tags tell your browser how to interpret the text and display it. For example, < h2 > is a mid-section header. It means the text is bigger, but not the biggest. The following is an example of HTML tags and text:


                <h2>A Mid Header</h2>
                <div> This is a division. It adds a section to your page. <em>This is emphasized text.</em></div>
                Pusheen: <img src="https://i.pinimg.com/originals/cb/f6/8b/cbf68beb6ad6bad1e301a442b6432a45.gif"/>
